# Lesson 3 – LED

## 📘 Overview
In this lesson, I learned how to change the brightness of an LED by using different values of resistor(220 ohm, 1k ohm and 10k ohm).

## 🔧 Components Used
- Arduino Uno R3 Board × 1
- 5mm red LED × 1
- 220 ohm resistor × 1
- 1k ohm resistor × 1
- 10k ohm resistor × 1
- M-M wires × 2
