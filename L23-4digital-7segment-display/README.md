# Lesson 23 – Four Digital Seven Segment Display

## 📘 Overview
In this lesson, I learned how to use a 4-digital 7-segment display and how to handle it.

## 🔧 Components Used
- Arduino Uno R3 Board × 1
- M-M wires × 23
- 74HC595 IC × 1
- 220 ohm resistor × 4
- 4-Digit 7-Segment Display × 1
