# Lesson 5 – Digital Inputs

## 📘 Overview
In this lesson, I learned to use push buttons with digital inputs to turn an LED on and off.

## 🔧 Components Used
- Arduino Uno R3 Board × 1
- M-M wires × 7
- 5mm red LED × 1
- 220 ohm resistors × 1
- push switch × 2
