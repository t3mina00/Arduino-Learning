# Lesson 18 – Serial Monitor

## 📘 Overview
In this lesson, I learned how to use a serial monitor to send a message to an Arduino.

## 🔧 Components Used
- Arduino Uno R3 Board × 1
- LED × 8
- 220 ohm resistors × 8
- M-M wires × 14
- 74hc595 IC × 1
